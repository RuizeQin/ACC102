## Project Overview
This project uses Python and WRDS (Wharton Research Data Services) data to analyse the return and volatility of the S&P 500 index. The goal is to help retail investors understand market risk and return trends from 2020 to 2026.



## Data Source
 Database: WRDS – CRSP Monthly Stock Index (crsp.msi)
 Index: S&P 500 (^GSPC)
 Period: 2020-01-01 to 2026-04-01
 Frequency: Monthly
 Access Date: April 2026



## Python Workflow
1. Connect to WRDS database
2. Download monthly index return data
3. Data cleaning and preprocessing
4. Calculate monthly returns
5. Compute 6‑month rolling volatility
6. Output descriptive statistics
7. Visualise return and volatility trends



## Main Results
- Average monthly return: Positive over the period
- Higher volatility in 2020 (COVID‑19) and 2022 (rate hikes)
- Automatic stabilisation of long‑term market trends
